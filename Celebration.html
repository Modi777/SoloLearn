Movie File
Merger

Movie File Merger supports the selective combination of video collections, movies, series, or documentaries. It can be used also for other types of collections, but it's optimized for videos.


This is an 'advertising' page with the purpose to show the free technologies used for my open source project in action... Movie File Merger.

Movie File Merger points out bad names of movies, documentaries, or episodes so that they can be renamed with specialized renaming apps. Unrelated files can be moved to the recycle bin. Videos can be sucked out of sub-folders and moved to the according folders.

Movie File Merger remembers all videos you have in different categories, like Movies, Documentaries, Series, Adult, Miscellaneous, or Clips, and which videos you don't want to have anymore. You'll never copy the same videos again and again.

How do I get files from Windows Explorer into Movie File Merger?

Just use drag and drop the files or a top level folder.

  • You can drag and drop files from Windows Explorer onto Movie File Merger.
  • One title is listed only once.
  • You can also drag a complete folder.

Note: Turn on the subtitles to see written instructions in the captions.

How do I search for movies on torrent sites?

Take a movie you want to download and drop it on the Torrents drop area.

  • Select the torrent from where to download.
  • Take a movie and drop it on the Torrents drop area.
  • Select the according torrent download.
  • The torrent download is added to your torrent program.

Note: Turn on the subtitles to see written instructions in the captions.

How do I search for movies information on the web?

Take any list entry and drop it on the Wen Info drop area.

  • Select the web page from where you want to get the information.
  • Take some movies and drop them on the Web Info drop area.
  • Each movie is shown in a separate tab in the browser.

Note: Turn on the subtitles to see written instructions in the captions.

How do I cleanup downloads from torrent sites?

You clean up the downloaded torrent files by deleting the garbage, sucking them out of the folder and renaming them.

  • Scan the completed downloads folder.
  • Delete garbage files.
  • Suck files out of the folders.
  • Rename the files with file renaming apps.

Note: Turn on the subtitles to see written instructions in the captions.

How do I start playing a movie from MFM?

Just drag the movie on the Play drop area.

  • Drop your existing movies in the Import list
  • Drag a movie on the Play drop area.

Note: Turn on the subtitles to see written instructions in the captions.

How do I rip my friend's movie collection?

Use Just Rip It to copy all your friend's movies, which you do not have at the moment. Garbage movies are ignored.

  • Drop your friend's movies in the Import list.
  • Click on Just Rip It.
  • Click on Start Process

Note: Turn on the subtitles to see written instructions in the captions.

How do I update my existing movie collection?

Use Just Scan It to add all movies in all top level folders containing Movies in the name.

  • Connect the hard disks with your Movies folders.
  • Click on Just Scan It.
  • The movies are updated and the result is shown in the log file.

Note: Turn on the subtitles to see written instructions in the captions.
